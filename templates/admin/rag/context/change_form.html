Context Type Guide

  • PDF: Upload PDF documents that will be automatically parsed and chunked
  • Markdown: Create content using Markdown syntax with direct editing capabilities
  • FAQ: Create question-answer pairs for frequently asked questions

Workflow Steps

  1. Select "PDF" as context type
  2. Fill in name and description
  3. Upload PDF file using the file upload field
  4. Save - the PDF will be automatically parsed and chunked
  5. Review the generated chunks in the Context Items section
  1. Select "Markdown" as context type
  2. Fill in name and description
  3. Enter your markdown content in the Markdown Content field
  4. Save - the content will be automatically chunked by sections
  5. Review the generated sections in the Markdown Sections
  1. Select "FAQ" as context type
  2. Fill in name and description
  3. Save the context first
  4. Use the "Add Q&A Pair" action to add question-answer pairs
  5. Review Q&A pairs in the Q&A Pairs section
